# Academic Databases and Sources by Discipline

Use this reference to select appropriate databases for the research topic. Prioritize free/open databases and APIs for programmatic access. When institutional access is available, supplement with subscription databases.

## General Multidisciplinary (start here for any topic)

| Database | URL | Access | API | Notes |
|----------|-----|--------|-----|-------|
| **OpenAlex** | https://openalex.org/ | Free | REST API, 100K req/day | 474M+ works. Best free alternative to Scopus/WoS. Covers all disciplines. |
| **Semantic Scholar** | https://www.semanticscholar.org/ | Free | REST API (rate limited) | 205M+ pubs. AI-generated TLDRs, citation graphs, embeddings. |
| **Google Scholar** | https://scholar.google.com/ | Free | No official API | Broadest coverage including grey lit. Use via web search. |
| **Crossref** | https://api.crossref.org/ | Free | REST API, no auth needed | 156M+ DOI records. Essential for metadata retrieval. |
| **Scopus** | https://www.scopus.com/ | Institutional | Elsevier APIs | 84M+ records. Strong non-English coverage. |
| **Web of Science** | https://www.webofscience.com/ | Institutional | Lite API | Science/Social Science/A&H Citation Indexes. Impact Factor source. |
| **JSTOR** | https://www.jstor.org/ | Institutional (some free) | DFR tools | Strong historical/archival coverage. 39K+ free research reports. |

## Recommended Search Strategy by Database Priority

For any literature review, search in this order:

1. **Start broad**: OpenAlex or Semantic Scholar (free, comprehensive, API-accessible)
2. **Add discipline-specific databases** from the sections above
3. **Check preprint servers** relevant to the field
4. **Search grey literature** sources if relevant to the research question
5. **Use Unpaywall/CORE** to find open access versions of paywalled papers
6. **Citation chain** from key papers using Semantic Scholar or OpenAlex citation graphs
